# Sizeable

The `sizeable` package is a type-only package for the sizes of the Vue Interface components.

## Installation

```bash
pnpm i @vue-interface/sizeable
```

```bash
yarn add @vue-interface/sizeable
```

```bash
npm i @vue-interface/sizeable
```

```bash
bun i @vue-interface/sizeable
```

## Basic Usage

For more comprehensive documentation and examples, please visit the [online documentation](https://vue-interface.github.io/packages/sizeable/).

# Textarea Field

The `textarea-field` component provides flexible and customizable textarea fields with customizable sizes and states.

## Installation

```bash
pnpm i @vue-interface/textarea-field
```

```bash
yarn add @vue-interface/textarea-field
```

```bash
npm i @vue-interface/textarea-field
```

```bash
bun i @vue-interface/textarea-field
```

## Basic Usage

```vue
<TextareaField v-model="value" label="Text Field" placeholder="Some placeholder"></TextareaField>
```

For more comprehensive documentation and examples, please visit the [online documentation](https://vue-interface.github.io/packages/textarea-field/).
